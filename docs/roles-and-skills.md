## Roles and responsibilities

**1. Backend Engineer**
- Designs, develops, and maintains microservices and APIs
- Implements business logic for services like Dispatch, Pricing, and Payment
- Ensures scalability, performance, and reliability of services
- Collaborates with frontend/mobile teams on API contracts
- Writes unit and integration tests
- Participates in code reviews and architectural decisions

**2. DevOps Engineer**
- Implements and maintains CI/CD pipelines for automated deployment
- Manages container orchestration (Kubernetes) and infrastructure as code
- Configures monitoring, logging, and alerting systems
- Ensures high availability and scalability of services
- Automates infrastructure provisioning and configuration
- Collaborates with development teams on deployment strategies

**3. Site Reliability Engineer (SRE)**
- Defines and monitors service level objectives (SLOs) and indicators (SLIs)
- Implements and manages incident response processes
- Performs capacity planning and load testing
- Works on improving system reliability and reducing toil through automation
- Conducts post-mortems and implements improvements
- Balances feature development with reliability requirements

**4. Data Scientist**
- Develops and optimizes algorithms for driver-rider matching
- Creates dynamic pricing models based on demand-supply analysis
- Builds predictive models for ETA and traffic patterns
- Analyzes large datasets to identify patterns and optimize operations
- Collaborates with engineering teams to implement models in production
- Validates model performance and makes iterative improvements

**5. Mobile Engineer (iOS/Android)**
- Develops and maintains the Yandex Go mobile applications
- Implements UI components following design specifications
- Integrates with backend APIs and handles network communication
- Implements push notification handling
- Optimizes app performance and battery usage
- Ensures compatibility across different devices and OS versions
- Publishes apps to App Store and Google Play Store

## Common skills across roles

**Technical Skills:**
- **Programming Languages**: Proficiency in languages like Java, Kotlin (Android), Swift (iOS), Python, Go, or Scala
- **Cloud & Infrastructure**: Experience with cloud platforms (Yandex Cloud, AWS, GCP), Kubernetes, Docker, and infrastructure as code (Terraform)
- **Databases**: Knowledge of both SQL (YDB, PostgreSQL) and NoSQL (Redis, ClickHouse) databases
- **API Development**: Experience with REST, gRPC, and GraphQL APIs
- **Monitoring & Observability**: Familiarity with tools like Prometheus, Grafana, Jaeger, ELK stack
- **CI/CD**: Experience with Jenkins, GitLab CI, GitHub Actions, or similar tools
- **Message Brokers**: Understanding of Kafka, RabbitMQ, or other event streaming platforms
- **Testing**: Knowledge of unit testing, integration testing, and automated testing frameworks

**Soft Skills:**
- **Collaboration**: Ability to work in cross-functional teams (engineering, product, design, business)
- **Communication**: Clear communication of technical concepts to both technical and non-technical stakeholders
- **Problem-solving**: Analytical thinking and systematic approach to troubleshooting
- **Agile Methodologies**: Experience with Scrum, Kanban, or other agile frameworks
- **Documentation**: Ability to create and maintain technical documentation

**Domain-Specific Knowledge:**
- **Microservices Architecture**: Understanding of distributed systems patterns
- **Geospatial Systems**: Knowledge of geospatial data structures and algorithms (for relevant roles)
- **Payment Systems**: Understanding of PCI DSS compliance and payment processing (for payment-related roles)
- **Mobile Development**: Knowledge of platform-specific guidelines and best practices (for mobile roles)
- **Data Engineering**: Understanding of ETL processes and data pipelines (for data-focused roles)
